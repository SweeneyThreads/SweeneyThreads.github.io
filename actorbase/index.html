How it works

Actorbase is written in Scala with the Akka toolkit and runtime.
It is based on the actor model, every actor talk each other only with messages.
There are several types of actor, each one reacts in in different ways to messages.
An actor can have children, if a child dies the parent reacts.

With this in mind we built a complete distributed key-value database and released it with MIT licence.
